# Mini Loyalty Program Integration

Mini Loyalty Program for demo purpose

## Run

```bash
go run ./cmd/server
```

Open `http://localhost:8080`.

## Demo Walkthrough

1. Click **Generate sandbox** on the prefilled Acme Cosmetics setup form.
2. Open the generated **Rewards page**.
3. Click **Log In**, select Olivia Chen, and continue.
4. Redeem the `$10 off` reward.
5. Open the generated **Storefront** and buy the demo product.
6. Open **Admin/debug** and refresh.

## Stack
- Golang + HTML Templating Engine, Vanilla JS.
